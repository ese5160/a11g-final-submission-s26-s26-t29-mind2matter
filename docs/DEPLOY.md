# GitHub Pages Deployment

This folder is the source for the A11 public project website.

Public source repository:

`https://github.com/ese5160/a11g-final-submission-s26-s26-t29-mind2matter`

Published GitHub Pages website:

`https://ese5160.github.io/a11g-final-submission-s26-s26-t29-mind2matter/`

## Deploy From GitHub

1. Commit and push the repository with `docs/index.html`, `docs/styles.css`, and any final files under `docs/assets/media/`.
2. Open the repository on GitHub.
3. Go to **Settings -> Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the submission branch, then choose `/docs` as the publishing folder.
6. Save and wait for GitHub Pages to publish the site at the URL above.
